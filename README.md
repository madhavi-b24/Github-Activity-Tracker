# GitHub Activity Tracker 📊

A Streamlit-based web application that generates a dynamic PDF report of a GitHub user's activity using the GitHub REST API.

## 🚀 Features
- Fetch GitHub profile and repository data
- Analyze total commits across repositories
- Identify most recent commit date
- Generate downloadable PDF activity report
- Secure token-based authentication

## 🛠 Tech Stack
- Python
- Streamlit
- GitHub REST API
- FPDF

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
